# JCIEスクロールテリングプロジェクト - ドキュメント

## 概要

グローバルファンド対策（AIDS・結核・マラリア）の効果をデータとストーリーで伝えるスクロールテリングプラットフォームのドキュメント集です。

## 📚 ドキュメント一覧

### 🎯 システム設計

#### [論理名ステップ管理システム](LOGICAL-NAME-SYSTEM.md)
- **目的**: 数値インデックスから論理名ベースのステップ管理への移行
- **対象**: main.js, HTML, 設定ファイル
- **成果**: ステップ追加・削除時の整合性自動保証

#### [設定ファイル分割システム](CONFIG-SPLIT-SYSTEM.md)
- **目的**: 機能別・環境別設定ファイル管理
- **対象**: config/ ディレクトリ全体
- **成果**: 保守性と柔軟性の向上

#### [統合設定システム](CONFIG-SYSTEM.md)
- **目的**: アニメーション・色・レイアウト値の外部設定化
- **対象**: 全コンポーネント
- **成果**: デザイン調整の簡素化

### 🎨 コンポーネント仕様

#### [ChartManager](ChartManager-README.md)
- **機能**: チャート表示管理
- **対応**: 折れ線・棒・円・グリッドチャート
- **特徴**: レスポンシブ対応、トランジション

#### [ChartTransitions](ChartTransitions-README.md)
- **機能**: チャート間のスムーズなトランジション
- **対応**: Object Constancy、段階的更新
- **特徴**: データ追加・削除時の自然なアニメーション

## 🔧 技術スタック

### フロントエンド
- **D3.js**: データ可視化・チャート描画
- **Scrollama.js**: スクロールベースのナラティブ
- **Tailwind CSS**: ユーティリティファーストCSS
- **Vanilla JavaScript**: フレームワーク非依存

### データ・設定
- **CSV**: チャートデータ
- **JSON**: 設定ファイル、地図データ
- **GeoJSON**: 地理データ

### 開発・ビルド
- **Python HTTP Server**: 開発サーバー
- **Browser DevTools**: デバッグ・テスト
